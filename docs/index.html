Harden your package managers
against supply chain attacks.

One command to scan npm, pnpm, bun, and uv configs for security best practices. Interactive TUI to apply fixes. Zero dependencies. Cross-platform.

$ cargo install depsguard
$ brew install arnica/tap/depsguard

What it protects

📦

npm

  • min-release-age=7 — delays new versions by 7 days
  • ignore-scripts=true — blocks malicious install scripts
⚡

pnpm

  • minimumReleaseAge — workspace-level release delay
  • trustPolicy: no-downgrade — blocks provenance downgrades
  • strictDepBuilds: true — fails on unreviewed scripts
🥟

bun

  • install.minimumReleaseAge=604800 — 7-day delay (seconds)
🐍

uv

  • exclude-newer — excludes recently published packages

How it works

1

Scan

Detects installed package managers and shows what's configured, what's missing, and what needs fixing.

2

Select

Interactive TUI lets you pick which fixes to apply. Arrow keys to navigate, space to toggle, enter to apply.

3

Done

Fixes applied with automatic backups. Use --restore to undo anytime.

Install

Cargo (Rust)
cargo install depsguard
Homebrew (macOS / Linux)
brew install arnica/tap/depsguard
Scoop (Windows)
scoop bucket add arnica https://github.com/arnica/scoop-bucket && scoop install depsguard
Download binary (Linux / macOS / Windows)
curl -fsSL https://github.com/arnica/depsguard/releases/latest/download/depsguard-$(uname -m)-$(uname -s | tr A-Z a-z).tar.gz | tar xz
