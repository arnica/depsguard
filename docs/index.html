Harden your package managers against supply chain attacks.

One command to scan npm, pnpm, bun, and uv configs for security best practices. Interactive terminal UI to apply fixes. Zero dependencies. Cross-platform. Free, MIT licensed.

Download binary

Install

Homebrew tap (macOS first, Linux supported)

brew tap arnica/depsguard
brew install depsguard

Recommended for macOS; works on Linux with Homebrew too.

What it protects

📦

npm / pnpm (.npmrc)

  • min-release-age=7 — delays new versions by 7 days
  • ignore-scripts=true — blocks malicious install scripts
⚡

pnpm (workspace)

  • minimumReleaseAge — workspace-level release delay
  • trustPolicy: no-downgrade — blocks provenance downgrades
  • strictDepBuilds: true — fails on unreviewed scripts
🥟

bun

  • install.minimumReleaseAge=604800 — 7-day delay (seconds)
🐍

uv

  • exclude-newer — excludes recently published packages

How it works

1

Scan

Detects installed package managers and shows what's configured, what's missing, and what needs fixing.

2

Select

Pick which fixes to apply interactively. Arrow keys to navigate, space to toggle, enter to apply.

3

Done

Fixes applied with automatic backups. Use --restore to undo anytime.
